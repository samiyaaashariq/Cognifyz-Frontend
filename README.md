## 📌 About the Project
This is a modern and responsive frontend web application built using **HTML, CSS, and JavaScript**. It demonstrates real-world frontend development skills including DOM manipulation, responsive design, and theme switching functionality (light/dark mode).

The goal of this project is to improve UI/UX understanding and build interactive user interfaces using vanilla web technologies.

---

## 🎯 Features
- Fully responsive design for all devices  
- Light / Dark mode toggle functionality  
- Clean and modern UI  
- Interactive JavaScript functionality  
- Smooth user experience  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
